  # DeskPet  =^._.^=

  macOS 데스크탑 위에 올려두는 귀여운 클립보드 매니저 =^._.^=

  ## 소개
  - 문서/이미지/파일 작업을 더욱 편리하게 도와주는 앱
  - 복사한 텍스트 자동으로 저장
  - 여러 개의 텍스트 저장, 복사 가능
  - 드래그 앤 드롭으로 이미지/파일을 빠르게 관리

  ## 설치

  1. [Releases](링크) 에서 DeskPet.dmg 다운로드
  2. DMG 열고 DeskPet.app을 Applications 폴더로 드래그
  3. 처음 실행 시 우클릭 → 열기 (macOS 보안 경고 우회)
